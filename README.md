# 🚀 AI Smart Quiz Platform

An AI-powered Full Stack Quiz Management System built using **Java, Spring Boot, React.js, MySQL, JWT Authentication, and Groq AI**.

The platform enables administrators to create quizzes manually or generate AI-powered multiple-choice questions, while students can attempt quizzes with a timer, view results, and compete on a leaderboard.

---

# 📌 Features

## 🔐 Authentication
- JWT Authentication
- Role-Based Login (Admin & Student)
- Secure REST APIs
- Password Encryption using BCrypt

---

## 👨‍💼 Admin Features

- Dashboard Analytics
- Category Management
- Question Management
- Quiz Management
- AI Question Generator (Groq AI)
- AI Question Saving
- View Student Results
- Leaderboard

---

## 👨‍🎓 Student Features

- Student Dashboard
- Attempt Available Quizzes
- Quiz Timer
- Submit Quiz
- Instant Result
- Roll Number Support
- Quiz History
- Performance Tracking

---

## 🤖 AI Integration

The application integrates **Groq AI** to automatically generate multiple-choice questions.

Admin can:

- Select Category
- Select Difficulty
- Select Number of Questions

Groq AI generates:

- Question
- Option A
- Option B
- Option C
- Option D
- Correct Answer

The generated questions can be saved directly into the MySQL database.
